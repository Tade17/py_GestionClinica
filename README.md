Aplicacion de escritorio para un clínica

Aplicación de escritorio desarrollada en Java con Apache NetBeans para la administración integral de una clínica. El sistema permite gestionar pacientes, médicos, horarios y agendamiento de citas, utilizando una base de datos PostgreSQL para persistencia de datos
Funcionalidades Principales:

-La aplicación permite a los administradores y médicos gestionar eficientemente las operaciones comunes diarias de la clínica, como:

✅Gestión de usuarios con diferentes roles (médico, administrador).

✅ Gestión completa (CRUD) de Pacientes y Médicos.

✅Asignación y gestión de turnos médicos con validación de horarios.

✅Gestión de citas médicas.

✅Gestión de pagos y servicios ofrecidos por la clínica.

Stack de Tecnologías
Lenguaje: Java

Base de Datos: PostgreSQL

IDE: Apache NetBeans

Librerías :JDBC
